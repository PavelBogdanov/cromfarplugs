
Opera Cache Browser v1.0
~~~~~~~~~~~~~~~~~~~~~~~~
  Плагин к файл-менеджеру FAR v2.0.

  Плагин создан для того,  чтобы из FAR просматривать содержимое кэша и список
закачек интернет-броузера Opera.

Установка
~~~~~~~~~
  Создайте  каталог  operac4  в  Far\Plugins и распакуйте  в  него  содержимое
архива. Перезапустите FAR.  Если вы запускаете FAR с ключом "-co",  то сначала
уберите его (ключ) из команды запуска FAR. После запуска ключ можно вернуть на
место.

Использование
~~~~~~~~~~~~~
  Установите курсор на нужном файле, нажмите F11, выберите из меню Opera Cache
Browser.
  Либо добавьте в меню пользователя команду opera:<путь к файлу>

Настройка
~~~~~~~~~
  Из  меню   FAR  выберите  "Параметры -> Параметры внешних модулей",  затем -
Opera Cache Browser. Появится окно конфигурирования.
  Более подробно о настройке плагина можно узнать из его справки.  Нажмите F1,
находясь в окне конфигурирования.

Удаление
~~~~~~~~
1 Закрыть все копии FAR.
2 Воспользовавшись Проводником удалить папку, в которую был установлен плагин.
